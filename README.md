## Getting Started

### Installation
1. Simply open `index.html` in a web browser
2. All features work without any server setup
3. Data is persisted in browser's localStorage
4. No installation required

### Navigation
- Use the dropdown menu to navigate between pages
- All pages are interconnected for seamless workflow
- Back buttons available on each page
- Login/Register links on all pages

### File Management Workflow
1. **Start at Main Page** - Generate URLs and access all features
2. **Register/Login** - Create an account or sign in
3. **Use File Manager** - Create folders, upload files, organize content
4. **Upload ZIP Folders** - Extract entire project structures
5. **Edit Code** - Open files in the editor with AI-powered error detection
6. **Monitor Activity** - Track users and view logs in the terminal
7. **Deploy Site** - Generate URLs from index.html for sharing

### Deployment
1. Create an `index.html` file in the File Manager root folder
2. Click "Deploy Current Site" on the main page
3. A unique URL will be generated
4. Share the URL to let others access your deployed site
5. The URL can be opened in any browser
